# Expense Tracker (Java)

## Description:
This is a simple Java project to track daily expenses.  
It helps users to add expenses, view them, and see the total amount spent.

## Features:
- Add new expense
- View all expenses
- Show total expense

## Tools Used:
- Java
- ArrayList
- Scanner

## How to Run:
1. Open terminal or command prompt  
2. Compile the program: javac ExpenseTracker.java  
3. Run the program: java ExpenseTracker
